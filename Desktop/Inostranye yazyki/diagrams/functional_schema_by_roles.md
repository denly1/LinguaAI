# 🏗 Функциональная схема LinguaAI по ролям

## 📊 Общая структура системы

```
┌─────────────────────────────────────────────────────────┐
│                    ПОЛЬЗОВАТЕЛИ                         │
│  👤 Гость    📚 Пользователь    🔧 Менеджер    👑 Админ │
└────────────────────┬────────────────────────────────────┘
                     ↓
        ┌────────────────────────┐
        │   АУТЕНТИФИКАЦИЯ       │
        │  - Вход/Регистрация    │
        │  - Управление ролями   │
        │  - Сессия (localStorage)│
        └────────┬───────────────┘
                 ↓
    ┌────────────┴────────────┐
    ↓                         ↓
┌───────────┐           ┌─────────────┐
│ ОБУЧЕНИЕ  │←──────────│  СЛОВАРИ    │
│           │           │             │
│- Карточки │           │- Создание   │
│- Тесты    │           │- Импорт/    │
│- Игры     │           │  Экспорт    │
│- TTS      │           │- Поиск      │
└─────┬─────┘           └─────────────┘
      │
      ├──────→ ┌──────────────────┐
      │        │   ПРОГРЕСС       │
      │        │ - XP             │
      │        │ - Streak         │
      │        │ - Достижения     │
      │        │ - Лидерборд      │
      │        └──────────────────┘
      │
      └──────→ ┌──────────────────┐
               │ АДМИНИСТРИРОВАНИЕ│
               │ - Пользователи   │
               │ - Роли           │
               │ - Статистика     │
               └──────────────────┘
                       ↓
                ┌──────────────┐
                │ localStorage │
                │  (Хранилище) │
                └──────────────┘
```

## 1️⃣ РОЛЬ: ГОСТЬ 👤

### **Доступные модули:**
- ✅ Аутентификация (только вход/регистрация)

### **Функции:**

#### **Аутентификация:**
1. **Зарегистрироваться**
   - Ввод email, пароля, имени
   - Выбор языка интерфейса
   - Автоматическое присвоение роли "user"
   
2. **Войти в систему**
   - Ввод email и пароля
   - Проверка credentials
   - Создание сессии в localStorage
   - Перенаправление на Dashboard

### **Ограничения:**
- ❌ Нет доступа к обучению
- ❌ Нет доступа к словарям
- ❌ Нет доступа к прогрессу
- ❌ Нет доступа к администрированию

### **Связи с другими модулями:**
```
Гость → Аутентификация (регистрация) → localStorage (сохранение)
Гость → Аутентификация (вход) → Пользователь (повышение роли)
```

---

## 2️⃣ РОЛЬ: ПОЛЬЗОВАТЕЛЬ 📚

### **Доступные модули:**
- ✅ Аутентификация (полный доступ)
- ✅ Обучение (полный доступ)
- ✅ Словари (полный доступ)
- ✅ Прогресс (полный доступ)

### **Функции:**

#### **Аутентификация:**
1. Выйти из системы
2. Редактировать профиль (имя, аватар, язык интерфейса)
3. Сменить пароль
4. Удалить аккаунт

#### **Обучение:**
1. **Просмотр карточек**
   - Листание карточек
   - Переворот (термин ↔ перевод)
   - Фильтр по статусу (новые, изучаемые, усвоенные)
   
2. **Режим изучения (SM-2)**
   - Оценка знания (не знаю / знаю)
   - Автоматический расчёт интервалов повторения
   - Обновление статуса карточки
   
3. **Тест (4 варианта)**
   - Генерация вопросов
   - Выбор правильного ответа
   - Подсчёт результатов
   
4. **Режим сопоставления**
   - Выбор количества пар (4-10)
   - Сопоставление слово-перевод
   - Таймер и подсчёт ошибок
   
5. **Speed Round**
   - Выбор длительности (30/60/120 сек)
   - Быстрые ответы
   - Подсчёт серий (streak)
   
6. **TTS — озвучка слов**
   - Прослушивание произношения
   - Поддержка 8 языков
   
7. **Горячие клавиши**
   - Пробел — переворот карточки
   - ←→ — навигация
   - 1-4 — выбор варианта
   - Enter — следующий вопрос

#### **Словари:**
1. **Создать словарь**
   - Название, описание
   - Выбор языка (en, es, fr, de, it, pt, zh, ja)
   - Уровень сложности (A1-C1)
   - Теги, цвет обложки
   
2. **Добавить слово**
   - Термин, перевод
   - Фонетика (опционально)
   - Часть речи (опционально)
   - Примеры использования
   - Теги
   
3. **Поиск и фильтрация**
   - Поиск по названию, описанию, тегам
   - Фильтр по языку
   - Сортировка
   
4. **Экспорт CSV/JSON**
   - Экспорт одного словаря
   - Экспорт всех словарей
   - Скачивание файла
   
5. **Импорт CSV**
   - Загрузка файла
   - Парсинг данных
   - Добавление слов в словарь
   
6. **Удалить слово/словарь**
   - Подтверждение удаления
   - Каскадное удаление карточек

#### **Прогресс:**
1. **Просмотр прогресса**
   - Общий XP
   - Уровень
   - Прогресс до следующего уровня
   
2. **Получать XP за занятия**
   - +5 XP за каждую правильную карточку
   - +2 XP за каждую минуту занятия
   - Бонусы за серии
   
3. **Поддерживать серию (streak)**
   - Ежедневные занятия
   - Подсчёт дней подряд
   - Уведомления о риске потери серии
   
4. **Получать достижения**
   - "Первые шаги" — 10 карточек
   - "Марафонец" — 7 дней подряд
   - "Полиглот" — 3 языка
   - "Идеальная сессия" — 100% точность
   
5. **Просмотр таблицы лидеров**
   - Топ-100 пользователей
   - Фильтр по периоду (неделя/месяц/всё время)
   - Свой ранг
   
6. **Получать уведомления**
   - Toast-уведомления о +XP
   - Уведомления о достижениях
   - Напоминания о занятиях

#### **Настройки:**
1. **Сменить тему** (тёмная/светлая)
2. **Добавить язык обучения**
3. **Глобальный поиск** (Ctrl+K)
4. **Сбросить прогресс**

### **Ограничения:**
- ❌ Нет доступа к администрированию

### **Связи с другими модулями:**
```
Пользователь → Словари → Обучение (создание карточек)
Пользователь → Обучение → Прогресс (начисление XP)
Пользователь → Обучение → localStorage (сохранение)
Пользователь → Словари → localStorage (сохранение)
```

## 3️⃣ РОЛЬ: МЕНЕДЖЕР 🔧

### **Доступные модули:**
- ✅ Все модули Пользователя +
- ✅ Администрирование (частичный доступ)

### **Функции:**

#### **Все функции Пользователя +**

#### **Администрирование (только просмотр и блокировка):**
1. **Просмотр списка пользователей**
   - Таблица всех пользователей
   - Информация: имя, email, роль, XP, дата регистрации
   - Поиск по имени/email
   - Фильтр по роли
   - Сортировка
   
2. **Заблокировать пользователя**
   - Временная блокировка доступа
   - Причина блокировки
   - Разблокировка
   
3. **Просмотр статистики системы**
   - Общее количество пользователей
   - Распределение по ролям
   - Активность за период
   - Топ по XP
   - Популярные языки

### **Ограничения:**
- ❌ Не может изменять роли
- ❌ Не может удалять пользователей
- ❌ Не может очищать данные

### **Связи с другими модулями:**
```
Менеджер → Администрирование (просмотр) → localStorage (чтение)
Менеджер → Администрирование (блокировка) → Аутентификация (изменение статуса)
```

---

## 4️⃣ РОЛЬ: АДМИНИСТРАТОР 👑

### **Доступные модули:**
- ✅ Все модули Менеджера +
- ✅ Администрирование (полный доступ)

### **Функции:**

#### **Все функции Менеджера +**

#### **Администрирование (полный доступ):**
1. **Изменить роль пользователя**
   - Повышение: user → manager → admin
   - Понижение: admin → manager → user
   - Подтверждение изменения
   
2. **Удалить пользователя**
   - Полное удаление аккаунта
   - Удаление всех данных (словари, прогресс)
   - Подтверждение с вводом email
   
3. **Очистить данные обучения**
   - Сброс прогресса пользователя
   - Удаление всех карточек
   - Сохранение словарей (опционально)

### **Ограничения:**
- ❌ Не может удалить себя
- ❌ Не может понизить свою роль

### **Связи с другими модулями:**
```
Администратор → Администрирование (управление) → Аутентификация (изменение ролей)
Администратор → Администрирование (удаление) → localStorage (очистка данных)
Администратор → Администрирование (очистка) → Прогресс (сброс)
```

## 🔄 Наследование прав

```
Гость (2 функции)
  ↓ наследует
Пользователь (32 функции)
  ↓ наследует
Менеджер (35 функций)
  ↓ наследует
Администратор (38 функций)
```

**Принцип:** Каждая следующая роль наследует ВСЕ права предыдущей + добавляет свои.

## 💾 Взаимодействие с localStorage

### **Что сохраняется:**
1. **Аутентификация:**
   - Токен сессии
   - Данные пользователя (id, email, имя, роль)
   
2. **Словари:**
   - Массив всех словарей
   - Слова в каждом словаре
   
3. **Обучение:**
   - Карточки (id, слово, статус, интервал, следующая дата)
   - История занятий
   
4. **Прогресс:**
   - XP, уровень
   - Серия (streak)
   - Достижения
   - Статистика

### **Кто может изменять:**
- **Гость:** только создание новой записи при регистрации
- **Пользователь:** свои данные (словари, прогресс)
- **Менеджер:** свои данные + статус блокировки других
- **Администратор:** любые данные любого пользователя
