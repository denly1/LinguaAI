# 🔧 Руководство по админ-панели бота

## 📋 Обзор

**Теперь вы можете управлять ВСЕМ через бота!** Никаких файлов на сервере трогать не нужно.

### Что можно делать через админ-панель:

✅ **Редактировать все тексты** (приветствие, о клинике, услуги, FAQ, критическое сообщение)  
✅ **Управлять администраторами** (добавлять/удалять)  
✅ **Настраивать Bitrix24** (изменить webhook)  
✅ **Просматривать статистику** (пользователи, заявки, handoff)  
✅ **Управлять заявками** (активные, последние)  

---

## 🚀 Как открыть админ-панель

Отправьте команду боту:
```
/admin
```

Вы увидите главное меню админ-панели с кнопками:
- 📊 **Статистика**
- 📋 **Активные заявки**
- ✏️ **Редактировать тексты**
- 👥 **Управление админами**
- ⚙️ **Настройки бота**
- ❓ **Помощь**

---

## ✏️ Редактирование текстов

### 1. Изменить приветствие

1. `/admin` → **✏️ Редактировать тексты** → **👋 Приветствие**
2. Бот покажет текущий текст
3. Отправьте новый текст одним сообщением
4. Готово! ✅

**Пример:**
```
Добро пожаловать в нашу клинику! 
Мы работаем 24/7 и готовы помочь вам в любое время.
```

### 2. Изменить раздел "О клинике"

1. `/admin` → **✏️ Редактировать тексты** → **🏥 О клинике**
2. Отправьте новый текст
3. Готово! ✅

### 3. Изменить критическое сообщение

1. `/admin` → **✏️ Редактировать тексты** → **⚠️ Критическое сообщение**
2. Отправьте новый текст
3. Готово! ✅

**Пример:**
```
⚠️ В экстренных случаях звоните 112 или 103!
```

### 4. Редактировать услуги

1. `/admin` → **✏️ Редактировать тексты** → **🧾 Услуги**
2. Выберите услугу (например, "Вывод из запоя")
3. Выберите поле для редактирования:
   - Название
   - Краткое описание
   - Когда подходит
   - Длительность
   - Стоимость
4. Отправьте новый текст
5. Готово! ✅

### 5. Редактировать FAQ

1. `/admin` → **✏️ Редактировать тексты** → **❓ FAQ**
2. Выберите вопрос (1-5)
3. Отправьте новый вопрос и ответ в формате:
   ```
   Это анонимно?
   ---
   Да, мы гарантируем полную конфиденциальность.
   ```
4. Готово! ✅

## 👥 Управление администраторами

### Добавить нового админа

1. `/admin` → **👥 Управление админами** → **➕ Добавить админа**
2. Узнайте Telegram ID нового админа:
   - Используйте бота [@userinfobot](https://t.me/userinfobot)
   - Или попросите пользователя отправить `/start` вашему боту
3. Отправьте ID (например: `123456789`)
4. Готово! ✅ Новый админ получит доступ после перезапуска бота

### Удалить админа

1. `/admin` → **👥 Управление админами** → **➖ Удалить админа**
2. Отправьте ID админа для удаления
3. Готово! ✅

**⚠️ Нельзя удалить последнего админа!**

## ⚙️ Настройки бота

### Изменить Bitrix24 webhook

1. `/admin` → **⚙️ Настройки бота** → **🔗 Изменить Bitrix24 webhook**
2. Отправьте новый URL вебхука:
   ```
   https://your-domain.bitrix24.ru/rest/1/xxx/crm.lead.add.json
   ```
3. Готово! ✅

`/admin` → **📋 Активные заявки**

- Username клиента
- Сценарий

`/admin` → **📝 Последние заявки**

Показывает последние 5 заявок с описанием

## 💬 Ответы на заявки

### Как отвечать клиентам

1. Когда приходит заявка, в ней есть `[ID заявки: 123]`
2. Нажмите **REPLY** (ответить) на это сообщение
3. Напишите ответ
4. Бот автоматически отправит ваш ответ клиенту

**Пример:**
```
Админ получает:
┌─────────────────────
│ Новая заявка
│ Клиент: @user | id:123
│ Сценарий: Вызвать доктора
│ Описание: Запой
│ 
│ [ID заявки: 42]
└─────────────────────

Админ отвечает REPLY:
"Доктор выезжает через 20 минут"

Клиент получает:
💬 Ответ от администратора:

Доктор выезжает через 20 минут
```

## ❓ Часто задаваемые вопросы

### Q: Можно ли редактировать тексты без перезапуска бота?
**A:** Нет, изменения в `texts.yml` требуют перезапуска бота.

### Q: Что делать, если случайно удалил единственного админа?
**A:** Нельзя удалить последнего админа — бот не позволит.

### Q: Можно ли добавить больше 2 админов?
**A:** Да, можно добавить сколько угодно админов.

### Q: Как узнать свой Telegram ID?
**A:** Используйте бота [@userinfobot](https://t.me/userinfobot) — отправьте ему `/start`.

### Q: Нужно ли знать программирование для редактирования текстов?
**A:** **НЕТ!** Всё делается через бота, никакого кода трогать не нужно.

### Q: Где хранятся тексты?
**A:** В файле `texts.yml`, но вы редактируете их через бота.

### Q: Можно ли откатить изменения?
**A:** Да, просто отредактируйте текст снова через админ-панель.

## 🛡️ Безопасность

### Кто имеет доступ к админ-панели?

Только пользователи с ID из `ADMIN_CHAT_IDS` в файле `.env`.

### Как защитить админ-панель?

1. Не делитесь своим Telegram ID
2. Добавляйте в админы только доверенных людей
3. Регулярно проверяйте список админов через `/admin` → **👥 Управление админами**
