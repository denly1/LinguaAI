# LinguaAI — Инструкция по запуску

## 🚀 Быстрый старт

### 1. Запуск основного приложения

```bash
npm start
```

Приложение откроется на `http://localhost:3000`

### 2. Запуск email-сервера (для отправки чеков)

В отдельном терминале:

```bash
cd server
npm install
npm start
```

Email-сервер запустится на `http://localhost:3001`

## 🎓 Система курсов и доступа

### Как работает доступ:

1. **Гость** (по умолчанию при открытии сайта):
   - Может просматривать курсы
   - Может купить курс
   - НЕ может: использовать флеш-карточки, тьютор, игры, прогресс

2. **После покупки курса**:
   - Заказ отправляется менеджеру
   - Email-чек приходит на почту
   - После подтверждения менеджером — доступ ко всем функциям

3. **Отзывы о курсах**:
   - Оставлять отзывы могут только купившие курс
   - Рейтинг от 1 до 5 звёзд
   - Комментарий обязателен

---

## 🛠 Панель менеджера

Доступна по адресу `/manager` для ролей `manager` и `admin`.

### Функции:

1. **Курсы**
   - Создание новых курсов
   - Редактирование (название, описание, цена, тариф)
   - Публикация/снятие с публикации
   - Удаление

2. **Тесты**
   - Создание тестов с вопросами
   - Типы вопросов: выбор, ввод, да/нет
   - Настройка времени и проходного балла

3. **Игры**
   - Создание игр со словами
   - Типы: matching, speed, fill-blank, word-order

4. **Заказы**
   - Просмотр всех покупок
   - Подтверждение/отклонение заказов
   - Фильтрация по статусу

---

## 💬 Раздел Тьютор

Чат-помощник для советов по изучению языка. Требует запущенного бекенд-сервера и настроенного `OPENROUTER_API_KEY` в `server/.env`.

---

## 📊 Структура данных

Все данные хранятся в `localStorage`:

- **linguaai-auth** — пользователи и сессии
- **linguaai-app** — прогресс обучения, словари, карточки
- **linguaai-courses** — курсы, тесты, игры, покупки, отзывы

## 🔧 Технологии

- **Frontend:** React 18, TypeScript
- **Routing:** React Router v6
- **State:** Context API + useReducer
- **Storage:** LocalStorage
- **Email:** Nodemailer (SMTP Mail.ru)
- **Chat:** OpenRouter API (прокси через бекенд)
- **Icons:** Lucide React
- **Styling:** CSS Modules
