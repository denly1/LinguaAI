## 🎨 Пошаговая инструкция в draw.io

### **Шаг 1: Создать новую диаграмму**
1. Откройте https://app.diagrams.net
2. **Create New Diagram** → **Blank Diagram**
3. Назовите: `LinguaAI_Use_Case_Diagram`

### **Шаг 2: Включить библиотеку UML**
1. **More Shapes** (внизу слева)
2. Найдите **UML** → поставьте галочку
3. **Apply**

### **Шаг 3: Добавить акторов**
1. Из библиотеки **UML** перетащите **Actor** (человечек)
2. Разместите 4 актора слева, 1 справа
3. Подпишите их (двойной клик на фигуру)
4. Покрасьте через **Style** → **Fill Color**

### **Шаг 4: Нарисовать границу системы**
1. **Rectangle** (прямоугольник) из базовой библиотеки
2. Растяните на весь холст
3. **Style** → **Dashed** (пунктир)
4. Добавьте текст сверху: `«система» LinguaAI`

### **Шаг 5: Создать группы**
1. Используйте **Container** или **Swimlane** из UML
2. Создайте 7 контейнеров
3. Подпишите каждый
4. Покрасьте фон через **Fill Color**

### **Шаг 6: Добавить прецеденты (эллипсы)**
1. Из **UML** перетащите **Use Case** (эллипс)
2. Разместите 37 эллипсов в соответствующие группы
3. Подпишите каждый

### **Шаг 7: Соединить линиями**
1. **Association** (обычная линия): актор → прецедент
2. **Generalization** (треугольник): наследование акторов
3. **Dependency** (пунктир): `«include»` и `«extend»`

### **Шаг 8: Добавить надписи на связях**
1. Двойной клик на линию
2. Введите `«include»` или `«extend»`
3. Измените стиль текста: курсив, размер 9-10

### **Шаг 9: Добавить легенду**
1. Вставьте **Table** (таблица 2x7)
2. Заполните:
   - Столбец 1: цветные квадраты
   - Столбец 2: названия групп
3. Разместите в правом нижнем углу

### **Шаг 10: Финальная настройка**
1. Выровняйте элементы: **Arrange** → **Align**
2. Распределите равномерно: **Distribute**
3. Проверьте все связи
4. **File** → **Export as** → **PNG** или **PDF**

5. **Экспорт в высоком качестве**: File → Export as → PNG → 300 DPI
